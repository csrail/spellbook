Basic project setup to start developing.

```html

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <link rel="stylesheet" href="minimum.css">
    <link rel="stylesheet" href="style.css">

</head>

</body>
</html>
```
